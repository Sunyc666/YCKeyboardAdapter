# YCKeyboardAdapter

#### 介绍
各类三方键盘弹出高度适配，包括但不限于：搜狗输入法、百度输入法、讯飞输入法、章鱼输入法、QQ输入法等。

#### 安装教程

1.  可选方式一：pod 'YCKeyboardAdapter'
2.  可选方式二：将YCKeyboardAdapter文件导入项目
